# 30-Day SRE Learning Plan (End-to-End)

This folder contains a complete 30-day roadmap to learn SRE fundamentals through advanced production operations.

## Files

- [30-day-sre-learning-plan.md](30-day-sre-learning-plan.md) — detailed day-by-day curriculum
- [practice-tracker.md](practice-tracker.md) — daily scorecard, progress tracker, and review templates

## Outcomes by Day 30

- Build and use SLI/SLO/error-budget models
- Design actionable alerts with low noise
- Run incident response and postmortems
- Operate safe deployments (canary/rollback)
- Troubleshoot Kubernetes/network policy incidents
- Build reliability runbooks and on-call workflows
- Prepare for SRE interviews with scenario-based practice

## Suggested Daily Time

- Weekdays: 2–3 hours
- Weekends: 3–4 hours (labs + review)

## Recommended Order

1. Complete [30-day-sre-learning-plan.md](30-day-sre-learning-plan.md) day by day
2. Fill [practice-tracker.md](practice-tracker.md) daily
3. Do weekly review every Day 7, 14, 21, 28
4. Use Days 29–30 for final capstone + mock interview
