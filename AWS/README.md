# AWS SRE Topics

> AWS-first reliability, observability, delivery, and operations learning path.

## What this folder covers

This folder splits the broader AWS SRE material into smaller learning docs. Each topic includes:
- what it is,
- why it matters,
- an AWS-focused workflow,
- and a graphical Mermaid flowchart.

## Suggested Reading Order

1. Start with SLOs and error budgets.
2. Learn observability and incident response.
3. Move into automation, CI/CD, and security.
4. Finish with data services, EKS, migration, and Git workflows.
