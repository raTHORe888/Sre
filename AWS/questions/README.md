# AWS SRE Topic Q&A

> Interview-grade question and answer sets for each AWS SRE topic. Every file has **exactly 10 questions** chosen to mirror what is commonly asked in SRE, DevOps, and Cloud Reliability interviews. All content is sourced from **public references only**: AWS documentation, AWS Well-Architected Framework, Google SRE Book, CNCF docs, and general public best practices.

## What this folder is for

Each file pairs with a topic doc in [Sre/AWS](../). The Q&A here:
- helps with **self-review and interview prep**,
- reinforces real operational thinking,
- and stays AWS-first.

The **IaC and automation** Q&A is intentionally deeper, since this is a major SRE focus area.

## How to use

- Read the related topic doc first.
- Try to answer each question out loud before reading the answer.
- Use the answers as a starting point, then adapt to your environment.
- For interviews, focus on **trade-offs**, **AWS specifics**, and **war stories** you can tie to each answer.
