# Linux and Infrastructure SRE

> A topic set focused on **Linux engineering, large-scale infrastructure, object storage, automation, hardware, capacity, and technical leadership**. Filtered from a real SRE job description and mapped to public learning resources only.

## Why this folder exists

Many SRE roles, especially in storage and infrastructure-heavy organizations, require deep Linux and hardware knowledge in addition to cloud skills. This folder complements [AWS/](../AWS/README.md) by covering:

- Linux at scale across thousands of bare metal and virtual machines.
- Kernel, filesystem, and storage tuning.
- Network and storage troubleshooting with classic Unix tools.
- Configuration management with **Chef** and **Ansible**.
- Clustered object storage operations.
- Hardware lifecycle and troubleshooting.
- Capacity planning and growth forecasting.
- Technical leadership and project management for SRE.
